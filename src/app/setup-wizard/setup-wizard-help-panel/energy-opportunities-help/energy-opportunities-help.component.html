Details about the energy efficiency opportunities found in this assessment can be added here.

Click " Add Opportunity" to add a new opporunity to the list.

If provided, an Implementation Cost will allow for payback calculations to be added to the report.

Non-Energy Benefits can be added to opportunities with the " Add NEB" button.

Clicking this button will give you two options for adding a new NEB:

  1. " Add Custom NEB" will add a free form Non-Energy Benefit to the opportunity where the user can name the NEB and enter a direct financial impact in $/yr.
  2. " Search NEBs Database" will open a modal with a list of all the Non-Energy Benefits that are currently in the tool.

Once a NEB is entered, "Key Performance Metric Impacts" can be added. These are impacts that the NEB has on performance metrics the company is tracking.

The " Search Key Performance Metrics" button will open a modal with a list of all performance metrics in the tool. If a metric is known to be typically associated with the NEB, it will be identified with an .

Details about the performance metric and the impact the associated NEB will have can then be entered on the screen.
